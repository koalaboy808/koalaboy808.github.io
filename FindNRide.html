Find'NRide:

Whatever your hunger, we provide the information to get you there

[Web Development] [Django] [Postgres] [Heroku] [Javascript]







Build a RESTful web application that utiziles good web architcture practice. You were free to choose a real-world problem of your choice given that you leveraged a set of current industry accepted technologies.



The web application (using Django, HTML/CSS/Javascript, Postgres, and Heroku) finds the best route to a food joint based on the user's keyword input. Query the Yelp API for the food item and provide the the user a list of options nearby. It then uses a combination of Google and Uber APIs to list a number of transporation methods and their routes.



I was actively involved in the entire full stack development process. I specficially tackled Django integration and handling the Google and Uber APIs.
